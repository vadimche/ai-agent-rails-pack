# Demo Task: Format Prices in the Product List

## User request

In the Vue product list, show prices with the euro symbol and two decimal places.

## Important context

This looks like a small UI task, but agents often over-edit frontend code: they may refactor components, change API contracts, alter data models, or update unrelated CSS.

## Goal of the demo

Show how the rails keep a simple UI formatting task small, local and reviewable.
